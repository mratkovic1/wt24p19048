2. zadatak:
CSS optimizacija:

Ako se koristi mnogo sličnih stilova za .nekretnina, .nekretnina.kuca, .nekretnina.pp i sl., možete razmisliti o zajedničkom stilu i specifičnim izmjenama samo kada je to potrebno. Na primjer, možete dodati zajedničke stilove za .nekretnina i onda samo promijeniti pozadinu za specifične klase:

Dugme (.detalji-dugme):

Iako je dugme u funkciji, može biti korisno dodati efekt pritiska (hover i active stanje) kako bi korisnici imali bolju povratnu informaciju. Na primjer:

css
Copy code
.detalji-dugme:hover {
    background-color: #ff6600;
}

.detalji-dugme:active {
    background-color: #cc7a00;
}
